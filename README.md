Installation
============

On your own (from scratch)
--------------------------
# $ npm i --save express socket.io mysql
# $ npm i --save-dev grunt grunt-contrib-jshint grunt-nodemon

Installing from this repo
-------------------------
# $ git clone git@github.com:arabesq610/socket-notification.git
# $ npm install
